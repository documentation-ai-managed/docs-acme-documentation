## Tutorial Structure

Tutorials follow a clear, actionable structure. You start with prerequisites, move through numbered steps, and end with troubleshooting.

<Steps>
  <Step title="Prerequisites" icon="check-circle">
    Review required tools and knowledge level.

    ```bash
    node --version
    npm --version
    ```
  </Step>
  <Step title="Setup" icon="download">
    Install dependencies.

    <CodeGroup tabs="npm,yarn">
    ```bash
    npm init -y
    npm install express
    ```
    ```bash
    yarn init -y
    yarn add express
    ```
    </CodeGroup>
  </Step>
  <Step title="Build" icon="code">
    Create your first server.

    ```javascript
    const express = require('express');
    const app = express();

    app.get('/', (req, res) => {
      res.send('Hello from ACME!');
    });

    app.listen(3000, () => {
      console.log('Server running on port 3000');
    });
    ```
  </Step>
  <Step title="Test & Deploy" icon="rocket">
    Run locally and deploy to your platform.
  </Step>
</Steps>
